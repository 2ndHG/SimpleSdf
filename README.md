# SimpleSdf

SimpleSdf aims to provide a simple solution to generate signed distance field (SDF) bitmap of given font and character, and outputs all information you need to work with an SDF shader.

The project is based on other people's hard work, all of them are distributed with MIT license, so is this project.

This project includes code referencing, merging and modifying from the following projects:

* [SharpMSDF](https://github.com/MutsiMutsi/SharpMSDF.git)
* [Typography](https://github.com/Tacodiva/Typography)
